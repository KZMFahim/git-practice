Objective:

The objective of this C++ program is to demonstrate the use of basic programming concepts such as variables, the string data type, output statements (cout), and program structure. 
It stores a student's personal and academic information and displays it in a clear, formatted manner on the console.


Outcome:

After executing this program, the user will be able to:

Understand how to declare and initialize string variables in C++.
Learn how to organize code using comments and sections.
Display formatted text and variable values using cout.
Create a simple Student Information System that presents student details in a readable format.
Gain basic experience in writing, compiling, and running a C++ program, making it suitable for practicing Git and GitHub version control.
